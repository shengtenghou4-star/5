# Working Rules

1. Only this repository may be modified during FENCHA work.
2. The completed M2 frozen-pilot result is immutable historical evidence.
3. Later tuning must be labeled M2.1 or later and must not overwrite M2 metrics.
4. Heavy workflows are manual-only and self-hosted.
5. No registration token, PAT, password, API key, or proxy credential may be committed.
6. Never force-push over local Codex work. Rebase or merge while preserving both sides.
7. Raw data, caches, virtual environments, and generated diagnostics remain local unless a small result artifact is intentionally committed.
