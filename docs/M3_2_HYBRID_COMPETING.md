# M3.2：总风险与离任路径分工模型

## 动机

前两轮历史盲测暴露出明显的能力分工：

- M3.1 的分层收缩模型在 1,110 个 2015+ 快照上将总离任 integrated Brier 从 0.013035 降到 0.013031，整体 skill 为 +0.000305；
- 原模拟模型对“是否离任”的概率较差，但在真实发生离任的观测中，条件路径 Brier skill 为 +0.241605，条件 log-loss skill 为 +0.152364，路径准确率从 36.21% 提高到 67.24%。

M3.2 因此不再要求同一个模型同时完成两项任务：

1. **总离任概率**：使用 M3.1 在 2010–2014 年选出的分层收缩配置；
2. **条件离任路径信号**：使用原结构模拟模型；
3. **概率协调**：沿用竞争风险层，将总离任概率精确分配到互斥路径，并保持跨期限单调和概率守恒。

## 固定配置

总风险模型沿用已经锁定的 M3.1 配置：

- 国家层、任期层和政府情境层全部启用；
- 任期桶 730 天；
- 国家、任期、情境收缩强度均为 400。

路径模型沿用 M3 竞争风险盲测中已经固定的结构模拟配置：

- `top_k=50`；
- `prior_strength=8`；
- `minimum_similarity=0.05`；
- IQR 数值缩放。

## 证据等级

这是**探索性组合实验**，不是新的独立确认性盲测。原因是两个组件各自在同一个 2015+ 留出块上的表现已经被观察过。结果用于验证系统集成是否符合预期、机制概率是否改善，不得被描述为完全未见数据上的独立发现。

## 运行

```bash
python -m fencha.m3_2_hybrid_cli \
  --holdout-start 2015-01-01 \
  --horizons 30,90,180,365 \
  --mechanisms post_election_transition,other_recorded_transition
```

输出：

```text
data/processed/m3_2_hybrid_competing_benchmark.json
```

重点比较：总离任 Brier、机制 Brier、条件路径 Brier/log loss/准确率，以及概率守恒误差。
