# M3.1：国家聚类不确定性检验

M3.1 的锁定留出集点估计将 integrated Brier 从 0.013035 降至 0.013031，skill 为 +0.000305。这个方向是正的，但幅度很小，不能仅凭一个平均数就把模型晋升为稳定主模型。

本阶段不再调参，只对已经锁定的 M3.1 配置做配对不确定性分析。

## 方法

1. 重放同一 2015+ 时间安全留出集；
2. 对每个完整的 30/90/180/365 天快照，记录历史基础率与 M3.1 的四期限等权 Brier；
3. 计算同一快照上的配对分数差；
4. 以国家为聚类单位进行 10,000 次有放回 bootstrap，整国重采样，保留同一国家内部的时间相关性；
5. 报告分数差和 Brier skill 的 95% 百分位区间，以及 bootstrap 中模型优于基础率的比例。

## 晋级规则

- 若配对差的 95% 上界小于 0，并且模型优于基础率的 bootstrap 比例至少为 95%，M3.1 可晋升为主总风险模型；
- 若区间跨过 0，则只保留为 challenger，即使点估计为正；
- 本检验只评估不确定性，不允许改变模型配置或评估样本。

## 命令

```bash
python -m fencha.m3_1_uncertainty_cli \
  --holdout-start 2015-01-01 \
  --horizons 30,90,180,365 \
  --replicates 10000 \
  --seed 20260714
```
